# Request Header Parser

Request Header Parser microservice implemented on Ktor.

## Instructions

To start this app:

- Build Application:

```
gradle build
```

- Running Ktor Application:

```
gradle run
```

or

```
java -jar build/libs/request-header-all.jar
```

- Run Ktor Test Application:

```
gradle test
```
