# Request Header Parser

Request Header Parser microservice implemented on Spring Boot.

## Instructions

To start this app:

- Build Application:

```
gradle build
```

- Running Spring Boot Application:

```
gradle bootRun
```

or

```
java -jar build/libs/requestheader-0.0.1-SNAPSHOT.jar
```

- Run Spring Boot Test Application:

```
gradle test
```
